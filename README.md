📊 Amazon Sales Analytics Dashboard
🚀 Project Title

Amazon Sales Data Analysis & Visualization

📌 Short Description / Purpose

This project focuses on analyzing Amazon sales data to uncover key business insights such as sales trends, profit distribution, and product performance. The dashboard helps stakeholders make data-driven decisions by providing a clear and interactive view of sales metrics over time.

🛠️ Tech Stack
📊 Power BI – Dashboard creation & visualization
🐍 Python (Pandas, NumPy) – Data cleaning & preprocessing (optional if used)
🗄️ SQL – Data querying & transformation (optional if used)
📁 Excel/CSV – Data storage
📂 Data Source
Public dataset (Amazon / Superstore-style sales dataset)
Contains fields like:
Order ID
Order Date
Product Name
Category
Sales
Profit
✨ Features & Highlights
📈 Sales Trend Analysis
Visualized sales over time to identify growth patterns and seasonal spikes
💰 Profit Insights
Tracked total profit (108K+) and identified high-performing products
🛒 Category-wise Sales Breakdown
Interactive treemap showing contribution of categories like Chairs, Phones, Storage, etc.
🏆 Top Products by Profit
Highlighted best-performing products driving maximum revenue
📊 KPI Cards
Total Orders: 3203
Total Sales: 725K+
Total Profit: 108K+
🎯 Interactive Dashboard
User-friendly visuals enabling easy filtering and business insights
